Heart Disease Classification Using Stacking Ensemble Framework

Issuer: IEEE i-PACT Conference (Communications in Computer and Information Science)

Novel stacking ensemble framework combining Restricted Boltzmann Machine with gradient boosting algorithms (XGBoost, LightGBM). Achieved 94.53% accuracy using RBM-MLP feature extraction paired with ensemble meta-learning on cardiovascular disease prediction. This research demonstrates how deep learning feature extraction can be effectively combined with traditional machine learning algorithms to improve medical diagnosis accuracy.
